# Smart Travel Planner

A **Flutter based Smart Travel Planner app** that helps users discover places, view weather details, explore images, and save favorite destinations with **offline support**.

---

##  Features

- Search places by city
- Discover popular destinations
- Live weather information
- Place images using Unsplash API
- Save favorite places (offline support with Hive)
- Works offline for saved data
- Clean, modern UI with smooth animations
- Scalable architecture (models, services, screens)

---

## Tech Stack

- **Flutter** (Dart)
- **Hive** – Offline storage
- **Dio** – API handling
- **OpenWeather API** – Weather data
- **Unsplash API** – Place images
- **Material 3 UI**
